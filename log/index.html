Patch Notes
Every game update since the launch of the public beta in January 2021
0.113.22025/02/10
  • Menus: Autohide mouse cursor + the [P] button immediately, show both on mouse movement
  • Lobbies: Fixed bug where P2 could not navigate the modding menu with their controller
  • Cabinet Mode: Increased idle timeout 60 > 180 seconds when NOT on freeplay
  • Cabinet Mode: Merged p1/p2 coin pool into shared coin pool
  • Cabinet Mode: Added options for making SFX/Music quieter during Attract
  • Cabinet Mode: Increased idle timeout 60 > 180 seconds when NOT on freeplay
  • Modding: Fixed bug where RCing while a projectile was spawning sub-projectiles would create duplicates
0.113.12025/02/03
  • Cabinet Mode: Added support for coin-op behavior
  • Cabinet Mode: Added Cabinet Options to Pause Menu, where you can configure the cost to play
  • Cabinet Mode: Hide Pause Menu button (P in bottom right) when cursor is inactive
  • Maintenance: Fixed copyright date
0.113.02025/01/29
  • Noodle: Frenzy is now unburstable until the final hit
  • Noodle: Fixed bug where you could spawn a 3rd bowl of Soup with a Whip Splash cancel
  • Beef: Added a damage scaling penalty to combos starting with Lemon or Air Lemon
  • Pork: Caber Toss will now auto tech enemy throws during startup
  • Onion: Added a damage scaling penalty to combos starting with Hopkick
  • Onion: Reduced hurtbox during Long Jump to better match the animation
0.112.52025/01/08
  • Update theme skins (Birthday, Summer, Halloween) to last the entire month
0.112.42024/11/12
  • FAQ: Updated multiple sections, added link to Steam page
  • Modding: AttackConfig can now set hit/block stun values to "skip" to prevent interrupting the opponent
  • Modding: Fixed bug where FrameConfig.behind ignored FrameConfig.mirrorImage
0.112.32024/10/23
  • Modding: InstallConfig.input fixed (never worked, whoops)
  • Modding: SequenceConfig.canTech added
  • Modding: TumbleConfig.juggle = "keep" added
  • Modding: TumbleConfig.restandStun added
0.112.22024/10/10
  • Summer skins: Fixed missing texture in Noodle's Whip Splash
  • Cabinet Mode: Added separate DipSwitches and editor in Debug Tools
  • Removed TODO page
0.112.12024/09/01
  • Summer skins: Fixed missing texture in Beef's uppercut
  • Versus CPU: Adjusted AI to reduce the number of random parries in neutral
  • Modding: Fixed bug with configs not loading when "projectiles" was left undefined
  • Modding: Fixed bug where FrameConfig.behind was ignored outside of particles
  • Modding: RemoteAnimationConfig.behind added
0.112.02024/07/09
  • Summer patch!
  • Six new Summer skins have been added (one for each character)
  • The skins will be freely playable online for the month of July, then locked to offline only starting August 1
  • Check out the free online tournament for a chance to permanently unlock summer skins! challonge.com/tlasummer24
  • Cabinet Mode: Enabled faster rematch during the win screen
  • Cabinet Mode: Changed Character Select to make it more obvious how to play local versus
  • Cabinet Mode: Disabled pressing Heavy to exit Character Select
0.111.02024/07/05
  • Animation: Garlic has been fully animated and polished
  • Animation: Garlic has new options for alternate colors
  • Cabinet Mode: Fixed bug where you could not access all the flavors
  • Cabinet Mode: Fixed bug with the Winner screen still accepting rematch inputs
  • Modding: Upgraded engine to 18
  • Modding: Garlic mods made before engine 18 will be upgraded to use new sprites
0.110.42024/07/02
  • Character Select: Fixed bug where you had to pick characters for both players despite it not mattering (playing online, tutorial, etc)
  • Cabinet Mode: Fixed bug where it was loading the incorrect default settings
  • Cabinet Mode: Stage now automatically changes between each match
  • Cabinet Mode: Hide Boss characters from Character Select even if unlocked / playing on Steam
  • Menus: Fixed bug where mouse cursor wouldn't auto hide unless you've moved it at least once
  • Menus: Decreased time for the mouse cursor to auto hide from 5 seconds to 3 seconds
0.110.32024/07/01
  • Gameplay: Fixed bug where missing asset prevented matches from loading
0.110.22024/07/01
  • Cabinet Mode added under Advanced Extras
  • Cabinet Mode, like Event Mode, locks the game into an alternative menu system that restricts game modes and is ideal for specific offline gameplay scenarios
  • If you wish to exit Cabinet Mode, you must use your mouse to click the P button in the bottom right
  • Modding: Updated docs to cleanup removed fields and fix typos
0.110.12024/06/09
  • Pork: Chop blockstun increased by 1, making it no longer punishable by Onion/Garlic Sweep
  • Onion: Flick can now cancel into Hopkick (L > S)
  • Cryptic Carb: Shrunk hitbox on Shockwave to make it easier to low profile
0.110.02024/06/09
  • Versus CPU: Added boss character "Cryptic Carb"
  • Modding: CancelConfig.onContact added for convenience
0.109.22024/05/14
  • Gameplay: Adjusted inescapable multi-hits (Rice Swat) to only KO on the last hit
  • Combo Trials: You can now verify that a hit was a "punish" or "antiair" (see Rice #4)
  • Combo Trials: Fixed bug where some combo progress persisted across position resets
  • Modding: AttackConfig.cannotKO added
0.109.12024/04/16
  • Ranked: Search range for "Similar Opponents" expanded 150 > 200

  • Steam: Disable clicking on the version modal to open up the status page
  • Modding: FighterStatsConfig.startMeter added
